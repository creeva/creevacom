Ok, I took all my blogs – with the exception of SNN<\/a>, and migrated them all into this page. Hopefully, this will give a more concise reading experience to my friends, family, and suffering folks that read my writings.<\/p>\n

I’m slowly adding all the old poetry and information from my past that I want archived online.<\/p>\n

A few things I’ve noticed though – I built this page on the new blogger beta – and a lot of services I was using in the past to post directly to my blog are not working. Hopefully, the support will be there soon – the best I’ve seen was Flickr which used the new authentication API to post directly to the blog.<\/p>\n

I know if you configure your browser for Flickr access it uses the new authentication API.
