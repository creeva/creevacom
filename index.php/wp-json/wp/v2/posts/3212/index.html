It’s been a long time since I’ve beeen excited about a new browser. Theoretically I’ve never been excited about a new browser that was announced. I remember being excited when AOL<\/a> resurrected Netscape<\/a> – but that turned into a flaming pile of poo and Netscape lost dominance being THE browser to use. Like many users at that time frame I used Internet Explorer<\/a> 5 and at the time it was best of breed, then a new challenger arose.<\/p>\n

The Mozilla foundation<\/a> announced they were taking the open source bits of the Netscape browser and making a new slimmer browser called Firebird. Because of issues of legal and copyright, Firebird was renamed to Firefox<\/a>. I’ve been using this browser since Firebird and I have had no reason to move to a different primary browser. I’ve tried Flock<\/a> and Safari<\/a>, there hasn’t been a sticky reason to keep using those over Firefox. I was excited, kind of, of the release of Firefox version 3. But that wasn’t a new and different browser, it was more of the same.<\/p>\n

With last nights announcement of Google’s New Chrome Browser<\/a>, but they put up a nice little web comic<\/a> that explains the features it offers. The security, privacy, performance enhancements alone make this a must watch for browser. WHen it is actually released later today, we’ll see how I feel then.<\/p>\n
